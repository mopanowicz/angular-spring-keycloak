# Frontend (Angular 20)

This is a minimal Angular 20 application that uses keycloak-js for authentication.

Prerequisites: Node 20+, npm/yarn, Angular CLI (optional).

Quick start:

1. cd frontend
2. npm install
3. npm start

Configuration:
- Edit src/environments/environment.ts to set KEYCLOAK_ISSUER, KEYCLOAK_REALM, KEYCLOAK_CLIENT
